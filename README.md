# Автоматизация UI- и API‑тестов для Кинопоиска

Проект содержит набор автоматизированных тестов для проверки функциональности веб‑сайта Кинопоиска через UI и API.

### Получение API‑ключа для тестов

Для запуска API‑тестов требуется действительный API‑ключ сервиса Кинопоиск.

#### Как получить ключ:

1. Перейдите на портал: `https://kinopoiskapiunofficial.tech/documentation/api/#/films/get_api_v2_2_films__id__relations`.
2. Пройдите авторизацию (https://kinopoiskapiunofficial.tech).
3. Найдите поле **API Key** или **Token**.
4. Скопируйте значение ключа. 

#### Настройка в проекте

1. Создайте файл `.env` в корне проекта (если его ещё нет).

## Функциональность


**UI‑тесты:**
* проверка кликабельности кнопок навигации;
* проверка работы фильтров при поиске;
* проверка функциональности поисковой строки;
* проверка наличия пояснительных текстов;
* проверка активности всех ссылок.

**API‑тесты:**
* поиск фильма по названию;
* получение отзывов к фильму;
* добавление фильма в список «Буду смотреть»;
* поиск по невалидному названию;
* поиск на другом языке.

## Настройка конфигурации
Откройте config.py.

Замените тестовые данные на актуальные:

при необходимости — BASE_URL и API_URL.

# Запуск тестов
## Все тесты - bash
pytest tests/ --alluredir=allure-results
## Только UI‑тесты - bash
pytest tests/test_ui.py --alluredir=allure-results
## Только API‑тесты - bash
pytest tests/test_api.py --alluredir=allure-results
## С подробной информацией - bash
pytest -v tests/ --alluredir=allure-results
# Запуск конкретного тестa - bash
pytest tests/test_api.py::TestKinopoiskAPI::test_search_movie_by_name -v

# Особенности реализации
Page Object Model — для UI‑тестов используется шаблон Page Object (pages.py).
Вспомогательные методы — API‑запросы централизованы в api_helper.py.
Allure‑отчёты — все тесты снабжены Allure‑разметкой для наглядных отчётов.
Конфигурация — тестовые данные вынесены в config.py для удобства поддержки.
